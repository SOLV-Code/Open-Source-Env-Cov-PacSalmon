### Key Points


**Index** | **Description**
-- | --
MEI | North Pacific Gyre Oscillation. Uses sea surface height variations as a proxy for nutrient fluctuations that affect primary production.
NPGO | North Pacific Index. Captures atmospheric pressure variations, which in turn are linked to sea surface temperature.
NPI | Oceanic Ni√±o Index. Uses SST anomalies from running average (detrended) for the eastcentral equatorial Pacific
ONI | Pacific Decadal Oscillation. North Pacific SST anomalies relative to global avg SST.
